yelp_localizedStringForKey:tableName:

Localize string with key.

- (NSString *)yelp_localizedStringForKey:(NSString *)key tableName:(NSString *)tableName

Parameters

key

String to localize

tableName

Name of strings file, defaults to “Localizable”

Discussion

Uses localized resource in order of: – locale identifier (language_region): “en_CA” – language: “en” – no locale: nil

If still not found will return the specified (default) value and finally will return the key as a last resort.

Localize string to the given locale.

Parameters

key

String to localize

tableName

Name of strings file, defaults to “Localizable”

locale

The locale to localize to

Discussion

Uses localized resource in order of: – locale parameter passed in (language_region): “en_CA” – language: “en” – device’s current locale

If still not found will return the key as a last resort.
