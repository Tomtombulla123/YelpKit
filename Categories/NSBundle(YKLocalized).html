yelp_localizedStringForKey:value:table:

Localize string with key.

- (NSString *)yelp_localizedStringForKey:(NSString *)key value:(NSString *)value table:(NSString *)tableName

Parameters

key

Key

value

Default localized string if key not found

table

Resource name; Defaults to Localizable

Discussion

Uses localized resource in order of: – locale identifier (language_region): “en_CA” – language: “en” – no locale: nil

If still not found will return the specified (default) value and finally will return the key as a last resort.
